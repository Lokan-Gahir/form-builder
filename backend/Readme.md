# Form Builder

## Tech Stack
- React (Frontend)
- Tailwind CSS
- Express + Node.js (Backend)
- MongoDB (Database)
- Multer (Image Uploads)

## Features
- Create forms with Categorize, Cloze, and Comprehension questions
- Upload images for form and questions
- Preview and fill forms
- Save responses in MongoDB
